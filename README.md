# Binary Tree Toolkit

## Overview
This project implements a Binary Search Tree (BST) toolkit in C++, including core operations and testing modules.

## Features
- BST insertion, search, and traversal
- Tree structure visualization
- Performance comparison tests
- Integration with Google Test (gtest)

## Project Structure
- src/ : implementation files
- include/ : header files
- gtest/ : unit and performance tests

## Build Instructions
```bash
mkdir build
cd build
cmake ..
make
